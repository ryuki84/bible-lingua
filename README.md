# Bible Lingua

듀오링고 스타일의 **영어 성경 학습 + 성경 암송** 루틴 서비스  
(1인 개발 / Windows 11 / VS Code 기반)

## Core Concept
- 배우고 (이해: 영어 성경)
- 외우고 (반복: 한글 암송)
- 매일 돌아오게 만드는 말씀 루틴

## Service Modes
- Learning Mode: 영어 + 한글 성경 학습
- Memory Mode: 한글 성경 암송 (단계형 UX)

## Tech Stack
- Frontend: Flutter
- Backend: Spring Boot (Java 17)
- IDE: VS Code
- AI: ChatGPT Plus, Gemini Code Assist, Codex CLI
