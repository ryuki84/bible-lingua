# UX Copy — Bible Lingua

## 공통 원칙
- 정답/오답 모두 “루틴 유지”에 초점
- 평가/훈계 톤 금지
- 짧고 따뜻하게

## Home
- 미완료: “오늘 3분만 해볼까요?”
- 학습 완료: “학습 완료. 이제 암송으로 이어가볼까요?”
- 암송 완료: “오늘 루틴을 완성했어요.”
- 둘 다 완료: “내일도 함께해요.”

## Learning
### 정답
- “좋아요. 흐름이 잡히고 있어요.”
- “잘했어요. 한 단계 올라갔어요.”

### 오답
- “괜찮아요. 다시 한 번 보면 됩니다.”
- “좋아요. 이미 반은 맞췄어요.”

## Memory (암송)
### Stage 1(읽기)
- “워밍업이에요. 천천히 읽어봐요.”

### 정답
- “말씀이 마음에 새겨지고 있어요.”
- “오늘 이 구절, 거의 외우셨어요.”

### 오답
- “괜찮아요, 다시 한 번!”
- “이미 절반은 기억하고 있어요.”

## Result
- 학습 완료: “오늘의 말씀 학습 완료.”
- 암송 완료: “오늘의 암송 완료.”
- 루틴 완료: “오늘 루틴을 완성했어요. 내일도 이어가요.”
