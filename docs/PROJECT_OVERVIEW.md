# Bible Lingua - Project Overview

## 한 줄 소개
Duolingo 스타일로 영어 성경을 짧게 학습하는 앱(Flutter) + API 서버(Spring Boot).

## 목표
- 매일 3~5분 학습으로 습관(스트릭) 만들기
- 복수 번역본을 초기부터 지원
- 최소 기능부터 실행 가능하게 만든 뒤 점진 확장

## 현재 상태
- GitHub 레포만 존재
- 로컬(E:\dev)에서 폴더 구조부터 다시 생성 예정

## MVP(1차 목표)
- Flutter: 오늘의 레슨 화면(목업) 실행
- Backend: health 체크 API 실행
- 이후: 오늘의 레슨 API로 연결
