# AI Tool Strategy

## Role Separation

### ChatGPT Plus
- 서비스 설계
- UX/아키텍처 사고 정리
- 복잡한 로직 설명
- 코드 리뷰 및 개선
- Codex 프롬프트 설계

### Gemini Code Assist (VS Code)
- 자동완성
- 함수/클래스 뼈대 생성
- Copilot 대체
- 무료 개인 사용

### OpenAI Codex CLI
- 에이전트형 AI
- 여러 파일 동시 수정
- 테스트/빌드 실행
- 기능 단위 구현

## Usage Rule
- Copilot 계열 = 타이핑 보조
- Codex = “일을 맡기는 AI 개발자”
- ChatGPT = 설계/의사결정 파트너

## License Safety
- 모든 도구 개인/일반 사용 플랜
- 영리 사용 가능
