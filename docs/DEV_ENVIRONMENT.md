# Development Environment Strategy

## Developer Context
- 1인 개발자
- 프론트엔드 + 백엔드 직접 개발
- Windows 11
- 개인 프로젝트 → 서비스 런칭 및 수익화 목표

- 비용 최소화

- 향후 확장/교체 가능
- 영리 사용 및 라이선스 안전성 확보
